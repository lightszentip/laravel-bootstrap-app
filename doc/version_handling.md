# Version Handling

## Version in Blade

````blade
@version
@version('full')
@version('compact')
````

## Show Version

````shell
php artisan version:show
````

## Update version

### Patch

````shell
php artisan version:patch   
````

### Minor

````shell
php artisan version:minor   
````

### Major

````shell
php artisan version:major   
````
